# Customer Segmentation Analysis

## Oasis Infobyte Internship
Domain: Data Analytics  
Task: Task 2

## Project Overview
This project performs customer segmentation using the K-Means clustering algorithm to identify different groups of customers based on their income and spending behavior.

## Dataset
Customer Segmentation Dataset

Features include:
- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn

## Methodology
1. Data Loading
2. Data Exploration
3. Feature Selection
4. Elbow Method
5. K-Means Clustering
6. Data Visualization

## Results
Customers were segmented into 5 clusters based on their income and spending behavior.

## Conclusion
Customer segmentation helps businesses understand customer patterns and improve marketing strategies.
